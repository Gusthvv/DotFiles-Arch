# ArchLinux-Hyprland Dotfiles

## Descriptions:
- **OS:** Arch Linux
- **Window Manager:** Hyprland
- **Bar:** Waybar
- **Laucher:** Rofi
- **Terminal:** Kitty
- **File Manager:** Pcmanfm
- **Fonts:** Nerd Fonts
- **Wallpaper Manager:** Hyprpaper
- **Audio System:** PipeWire
- **Browser:** Firefox
- **Task Manager:** Btop+
